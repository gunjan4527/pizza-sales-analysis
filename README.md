🍕 Pizza Sales Analysis Project
📌 Project Overview

This project focuses on analyzing pizza sales data to uncover meaningful business insights and trends. The goal is to understand customer behavior, identify top-performing products, and optimize sales strategies using data-driven decisions.

The analysis was performed using:

Excel → Data Cleaning & Preprocessing
SQL → Data Querying & Analysis
Power BI → Data Visualization & Dashboard
🛠️ Tools & Technologies
📊 Excel
🗄️ SQL
📈 Power BI
📂 Dataset Description

The dataset contains detailed pizza sales records, including:

Order ID & Date
Pizza Name & Category
Quantity Sold
Unit Price & Total Price
Pizza Size & Ingredients

This structured dataset helps analyze:

Sales performance
Customer preferences
Revenue trends
